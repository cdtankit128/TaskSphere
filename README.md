# Full Stack CI/CD Application

Full stack app with React + Spring Boot + Docker + GitHub Actions CI/CD pipeline.

## Tech Stack
Frontend uses React, backend uses Spring Boot, containerization is handled with Docker, and CI/CD is managed through GitHub Actions.

## Features
The project includes an automated build and test pipeline, Dockerized services, and deployment-ready CI/CD configuration.

## Project Structure
The `frontend/` directory contains the React application, the `backend/` directory contains the Spring Boot API, and `.github/workflows/` contains the CI/CD pipeline definitions.
